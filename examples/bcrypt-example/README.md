## Purpose: Demonstrates hashing and verifying passwords using the bcrypt library.

### Packages used:
bcrypt

### Functionality:

- Converts a plaintext password into bytes.
- Generates a bcrypt salt.
- Hashes the password using bcrypt.
- Verifies a correct password against the hash.
- Verifies an incorrect password against the hash.

### How to run the example :
```
chmod +x install_test_example.sh
./install_test_example.sh
```
