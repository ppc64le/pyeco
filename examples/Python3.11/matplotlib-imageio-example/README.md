## ✅ Program : Matplotlib + ImageIO Rendering & Validation Suite

### Purpose:
Provides a small image‑generation and validation pipeline using Matplotlib and ImageIO.
Each script independently creates visual artifacts (PNG, JPEG, GIF) and performs internal correctness checks.

### Packages used:
- matplotlib
- imageio
- pillow
- numpy

### Functionality:
- Loads configuration values from a YAML file.
- Renders static plots and saves them as PNG and JPEG.
- Generates animated GIFs using ImageIO’s Pillow backend.
- Creates contour plots and large tiled heatmap images.

### How to run the example :
```
chmod +x install_test_example.sh
./install_test_example.sh
```
