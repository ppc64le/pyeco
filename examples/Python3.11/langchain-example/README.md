## ✅ Program :  LangChain Mini Pipeline Demo

### Purpose:
Demonstrates a small LangChain workflow using prompts, fake LLM responses, structured parsing, and runnable composition.

### Packages used:
- langchain
- langchain-core
- langchain‑community
- numpy
- json
- pydantic

### Functionality:
- Creates prompt templates and passes them through a FakeListLLM.
- Generates a one‑sentence summary using a simple prompt chain.
- Produces structured JSON output and parses it into Python dicts.
- Builds a runnable sentiment‑classification chain using RunnablePassthrough.
- Prints results for summarization, JSON parsing, and classification.

### How to run the example :
```
chmod +x install_test_example.sh
./install_test_example.sh
```
