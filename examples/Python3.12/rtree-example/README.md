## ✅ Program :  Rtree Spatial Index Examples

### Purpose:
Demonstrates usage of the Rtree Python package for spatial indexing, including intersection queries, nearest neighbor search, object storage, index properties, and error handling.

### Packages used:
rtree numpy

### Functionality:
- Builds in‑memory Rtree indexes and inserts bounding boxes and points.
- Performs intersection queries with and without attached Python objects.
- Executes nearest‑neighbor (KNN) searches, including equidistant cases.
- Demonstrates duplicate ID insertion behavior.
- Shows safe use of index properties such as capacities and overlap factor.
- Handles invalid bounding boxes and argument errors gracefully.
- Attempts advanced vectorized APIs (intersection_v, nearest_v) when supported.w.

### How to run the example :
```
chmod +x install_test_example.sh
./install_test_example.sh
```
