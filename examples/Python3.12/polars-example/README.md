## ✅ Program: Polars DataFrame Library Comprehensive Example

### Purpose:
Demonstrates comprehensive usage of **Polars**, a blazingly fast DataFrame library implemented in Rust with Python bindings. The example covers data manipulation, transformations, aggregations, joins, lazy evaluation, window functions, and I/O operations.

### Packages used:
polars numpy pyarrow datetime

### Functionality:
- Creates DataFrames from dictionaries and numpy arrays
- Performs basic operations: select, filter, sort
- Applies expressions and transformations with column operations and aliases
- Executes aggregations with group by operations
- Handles date/time operations: extract year, month, day, weekday
- Performs string operations: case conversion, length, pattern matching, replacement
- Demonstrates various join types (inner, left, right)
- Utilizes lazy evaluation for query optimization
- Implements window functions for ranking and rolling operations
- Handles null values with fill and drop operations
- Performs pivot operations to reshape data
- Calculates statistical operations: mean, std, median, quantiles
- Reads and writes CSV files
- Validates all operations with comprehensive test coverage

### How to run the example:
```
chmod +x install_test_example.sh
./install_test_example.sh
```
