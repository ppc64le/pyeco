## ✅ Program :  Tree‑sitter Python AST Parsing Examples

### Purpose:
Demonstrates how to parse Python source code using tree‑sitter, inspect syntax trees, and extract structural information such as functions, classes, and imports.

### Packages used:
tree_sitter tree_sitter_python

### Functionality:
- Parses Python source code into an abstract syntax tree (AST).
- Traverses syntax trees using cursor‑based navigation.
- Extracts function and class definitions using tree‑sitter queries.
- Captures node names, parameters, and source ranges.
- Prints structured parsing results without using a test framework.

### How to run the example :
```
chmod +x install_test_example.sh
./install_test_example.sh
```
