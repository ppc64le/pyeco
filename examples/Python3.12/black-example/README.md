## Purpose: Formats Python code strings using the Black formatter.

### Packages used:
black

### Functionality:
- Creates a Black formatting mode.
- Formats a Python code string using Black.
- Handles formatting errors.
- Prints original and formatted code.

### How to run the example :
```
chmod +x install_test_example.sh
./install_test_example.sh
```
